Sistema de Gestión de Oficios
Una aplicación web Full-Stack para la creación, administración y seguimiento de oficios, desarrollada con Node.js y PostgreSQL.

Demo en Vivo: https://sistema-oficios.onrender.com

Tecnologías Utilizadas

Backend: Node.js, Express.js
Base de Datos: PostgreSQL 
Frontend: EJS (Motor de plantillas), HTML5, CSS, Bootstrap
Seguridad: Bcrypt (Hashing de contraseñas), Express-Session
Infraestructura: Render (Web Service + Managed PostgreSQL)

Funcionalidades
Autenticación: Registro e inicio de sesión seguro.

Roles de Usuario:

Usuario: Puede crear y visualizar oficios.
Administrador: Acceso total, gestión de usuarios, edición de oficios y búsqueda avanzada.
Gestión de Oficios: Generación automática de folio único y consecutivo.
Búsqueda Universal: Filtrado en tiempo real por destinatario, asunto, departamento o folio.
Seguridad: Protección de rutas mediante Middleware.
